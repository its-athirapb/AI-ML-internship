# Iris Flower Classification App

A Machine Learning web application built using Streamlit and Random Forest Classifier.

## Features
- Predicts Iris flower species
- Interactive sliders for input
- Displays confidence scores
- Built using Streamlit

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Joblib

## Dataset
Iris Dataset from Scikit-learn

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
